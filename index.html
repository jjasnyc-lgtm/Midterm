A redesign of Montefiore Square into a safer, greener, and more community-focused public space.

About the Project

This project explores how urban design can improve safety, comfort, and community use in Harlem’s Montefiore Square.

The Problem

The plaza currently lacks seating, greenery, and safe pedestrian circulation, making it underused.
